This is a minimal static webpage for insurance inquiries.

## Files

- `index.html` — webpage content
- `styles.css` — design/styling
- `assets/` — logo images

## Hosting

Upload the contents of this folder to any static host such as GitHub Pages, Netlify, Vercel, or your own hosting. After you get the final public URL, generate a QR code pointing to that URL.
